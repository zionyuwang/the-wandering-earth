# the-wandering-earth
The Wandering Earth
本闯关跑酷类小游戏以“趣味闯关+轻量化娱乐”为核心定位，聚焦用户碎片化娱乐需求，完整实现关卡设置、激励机制、广告设置三大核心产品需求，兼顾可玩性与商业变现，打造易上手、高留存的休闲跑酷体验。关卡设置采用梯度递进模式，共设计5差异化关卡，从基础新手关到高阶挑战关，逐步提升障碍物密度、移动速度及通关难度；激励机制：建立即时反馈机制，连续跳跃可有语言激励；广告设置遵循“不影响游戏体验”原则，采用自愿观看模式，观看激励广告可获得额外道具、复活机会或双倍奖励，在关卡失败、通关结算后弹出广告入口，避免强制弹窗，平衡用户体验与商业变现需求，整体提升游戏留存率与用户活跃度
